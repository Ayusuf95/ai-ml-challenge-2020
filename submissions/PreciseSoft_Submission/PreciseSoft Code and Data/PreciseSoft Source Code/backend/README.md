# README #

### How do I get set up? ###

* Dependencies

  **This project only support Python 3.X**

  ```python
  pip install --upgrade pip
  pip install -r requirements.txt
  ```

* Configuration

  Put model file under `gsa_server/resources/xlnet_6epoch_3e-5`

* To start server:

  ```python
  python -m gsa_server
  ```
